# React-Redux Budget Tracker
## Developer: Daniel Pillay 


## Installation: 
- Clone directory
- Run `npm install`
- run `npm run hot`

## Description: 
A budget tracker made with React-Redux. This application allows the user to add a budget and budget cost, along with deleting and updating the budget.
